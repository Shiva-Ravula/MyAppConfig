\# Smart Parking Management System



This project is a Smart Parking Management System developed for a semester project.



\## Features

\- User registration and login

\- Parking space management

\- Reservation system

\- Payment simulation

\- Admin dashboard



\## Run the project

1\. Install packages:

&#x20;  pip install -r requirements.txt



2\. Run the app:

&#x20;  python smart\_parking\_app.py



3\. Open in browser:

&#x20;  http://127.0.0.1:5000



This project was managed using Git and GitHub for configuration management.
